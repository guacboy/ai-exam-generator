# AI Exam Question Generator

A Python based desktop application that helps students study by generating exam questions directly from their own notes. The app uses the OpenAI API to create relevant questions and includes microservices for feedback, streak tracking, and suggestions.

## Features

- **Upload and parse student notes** (text files or typed input)
- **Generate exam questions** using the OpenAI API based on the uploaded content
- **Feedback analyzer** – evaluates user answers and provides improvement tips
- **Streak tracker** – monitors daily study streaks to encourage consistency
- **Suggestion service** – recommends which topics or chapters to review next
- **Simple GUI** built with tkinter for easy navigation and use

## Tech Stack

- Python 3
- tkinter for graphical user interface
- OpenAI API
- Custom microservices architecture for feedback, streaks, and suggestions
